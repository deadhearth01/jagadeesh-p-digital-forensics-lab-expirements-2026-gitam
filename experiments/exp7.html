EXPERIMENT 7

Live forensic case investigation using Autopsy

Aim

To conduct a comprehensive forensic investigation of a disk image using Autopsy digital forensics platform and analyze various artifacts.

Theory

Autopsy is a free, open-source digital forensics platform used by law enforcement, military, and corporate investigators to analyze hard drives and smartphones. It provides a graphical interface to The Sleuth Kit and other forensic tools.

Autopsy can analyze disk images in various formats (E01, DD, etc.) and automatically extracts artifacts such as web history, deleted files, EXIF data from images, email messages, and more. It uses ingest modules to process the data and present findings in an organized manner.

The tool creates a case file that organizes all evidence and analysis. Investigators can add tags, comments, and generate reports. Autopsy supports keyword searching across all files and can recover deleted files from unallocated space.

Tools Required

  • Autopsy Digital Forensics Platform (free download)
  • A forensic disk image (E01 or DD format)
  • Sufficient RAM and storage for processing
  • Windows operating system

Procedure

  1. Download and install Autopsy from the official website (sleuthkit.org)
  2. Launch Autopsy and create a new case (enter case name, number, examiner details)
  3. Add a data source - select the disk image file to analyze
  4. Configure ingest modules (select which analysis modules to run):
    • Recent Activity - browser history, downloads, recent documents
    • Hash Lookup - identify known files
    • File Type Identification - categorize files
    • Keyword Search - search for specific terms
    • Email Parser - extract email messages
  5. Start the ingest process and wait for analysis to complete
  6. Browse the Results tree to examine extracted artifacts
  7. Check Web History, Web Downloads, and Web Cookies
  8. Examine the Deleted Files section for recovered files
  9. Use keyword search to find specific content
  10. Tag important evidence and add examiner comments
  11. Generate an HTML report summarizing findings

Result

A comprehensive forensic investigation was successfully conducted using Autopsy. The tool automatically extracted various artifacts including web browsing history, deleted files, and metadata from images. The organized presentation of findings and the ability to generate reports makes Autopsy an effective tool for digital forensic investigations.
