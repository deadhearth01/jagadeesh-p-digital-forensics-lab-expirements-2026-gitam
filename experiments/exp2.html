EXPERIMENT 2

Study the steps for hiding and extracting any text file behind an image file/Audio file using Command Prompt

Aim

To hide a text file inside an image using Windows Command Prompt and then extract it back.

Theory

Steganography is the practice of hiding secret information inside a normal looking file. The word comes from Greek words meaning "covered writing". Unlike encryption which makes data unreadable, steganography hides the fact that secret data exists at all.

In Windows, we can use the COPY /B command to combine two files at the binary level. When we combine a JPEG image with a ZIP file, the resulting file still opens as a normal image. But if we change the extension to .zip, we can extract the hidden content.

This works because image viewers read only until the end of image marker and ignore any extra data appended after it.

Tools Required

  • Windows Command Prompt
  • WinRAR or 7-Zip
  • A JPEG image file
  • A text file with secret message

Procedure

Part A - Hiding the file:

  1. Create a text file called secret.txt with your message
  2. Compress secret.txt into secret.zip using WinRAR
  3. Keep both image.jpg and secret.zip in the same folder
  4. Open Command Prompt and navigate to that folder
  5. Run: copy /b image.jpg + secret.zip output.jpg
  6. The output.jpg file now contains hidden data

Part B - Extracting the file:

  1. Rename output.jpg to output.zip
  2. Open output.zip with WinRAR or 7-Zip
  3. Extract secret.txt from the archive
  4. Open the text file to see your hidden message

Result

The text file was successfully hidden inside the image file using the COPY /B command. The output image opened normally in any image viewer. By renaming the extension to .zip, the hidden text file was extracted successfully. This demonstrates a simple steganography technique that can be used to hide information.
