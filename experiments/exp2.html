Experiment 02

Hiding & Extracting Text Behind an Image/Audio File Using Command Prompt

🎯 Aim

To study the steps for hiding and extracting any text file behind an image file or audio file using Windows Command Prompt, and to understand steganographic concealment as a forensic artifact.

📖 Theory

Steganography is the practice of concealing information within a non-secret file or medium so that the existence of the hidden message is not apparent to an observer. Unlike cryptography, which makes data unreadable, steganography hides the very existence of the data.

In digital forensics, steganography is a significant concern because criminals often hide evidence inside seemingly innocent files such as holiday photos or music tracks. Forensic investigators must be able to both detect and extract such hidden content.

One of the simplest Windows-based steganography techniques exploits the COPY /B command, which performs a binary copy. By concatenating a ZIP archive (containing a text file) with a JPEG image, the resulting file is still viewable as an image but also contains the hidden archive. This works because JPEG viewers read only up to the end-of-image marker (FF D9) and ignore trailing data, while archive tools parse the ZIP end-of-central-directory signature.

Steganography CMD / COPY /B JPEG EOF ZIP Concatenation Anti-Forensics

🛠️ Tools Required

Windows Command Prompt (cmd.exe) — Built-in Windows shell used to execute the binary copy command.

WinRAR / 7-Zip — Archiving tools used to create the ZIP container for the hidden text file.

A JPEG image — The carrier file used to conceal the hidden data.

A text file — The secret message to be hidden.

📋 Procedure — Hiding Text in an Image

  1. Prepare the Files

    Place carrier.jpg (the host image) and secret.txt (your hidden message) in the same folder, e.g., C:\steg\.

  2. Archive the Text File

    Using WinRAR or 7-Zip, compress secret.txt into secret.zip and place it in the same directory.

  3. Open Command Prompt as Administrator

    Press Win + R, type cmd, and run as Administrator. Navigate to the working directory:

    cd C:\steg

  4. Concatenate Files Using COPY /B

    Run the binary copy command to merge the image and the ZIP archive:

    copy /b carrier.jpg + secret.zip output.jpg

    The resulting output.jpg appears as a normal image but contains the hidden archive.

📋 Procedure — Extracting Hidden Text

  1. Change the File Extension

    Rename output.jpg to output.zip in Command Prompt:

    ren output.jpg output.zip

  2. Extract the Archive

    Open output.zip with WinRAR or 7-Zip. The secret.txt file will be extracted, revealing the hidden message.

  3. Verify Content

    Open the extracted text file and confirm the hidden message is intact and unmodified.

📝 Observations & Result

The output JPEG file opened normally in image viewers without any visible difference from the original. The hidden ZIP archive was successfully extracted by renaming the file extension, and the original text message was recovered intact.

Result: Text was successfully hidden within an image file using the Windows CMD binary copy technique, and subsequently extracted — demonstrating a simple steganographic method used in real-world anti-forensic scenarios.
