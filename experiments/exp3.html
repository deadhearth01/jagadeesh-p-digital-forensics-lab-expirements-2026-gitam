Experiment 03

Extraction of EXIF Data from Image Files Using Exifreader Software

🎯 Aim

To extract and analyze the Exchangeable Image File Format (EXIF) metadata from image files using Exifreader software, and to understand how image metadata serves as forensic evidence.

📖 Theory

EXIF (Exchangeable Image File Format) is a standard that specifies the formats for images, sound, and ancillary tags used by digital cameras, smartphones, and scanners. EXIF metadata is embedded directly within JPEG, TIFF, and other image file formats.

EXIF data can contain a wealth of forensically valuable information:

  • GPS Coordinates — Latitude, longitude, and altitude where the photo was taken
  • Timestamps — Date and time of capture (DateTimeOriginal, DateTimeDigitized)
  • Device Information — Camera make, model, software version
  • Camera Settings — Aperture, shutter speed, ISO, focal length, flash status
  • Thumbnail — An embedded thumbnail that may differ from the modified image

In criminal investigations, GPS data embedded in photos has been used to locate suspects, place victims at crime scenes, and corroborate or contradict alibis. Notably, metadata can persist even when images are shared on social media (though many platforms strip EXIF on upload).

🛠️ Tools Required

Exifreader — A lightweight metadata viewer that displays all EXIF, IPTC, and XMP tags embedded in image files.

ExifTool by Phil Harvey — A command-line alternative for batch EXIF extraction and editing.

Sample JPEG images — Preferably taken with a smartphone with GPS enabled to demonstrate location metadata.

📋 Procedure

  1. Collect Sample Images

    Gather JPEG images taken with a camera or smartphone. Ensure at least one image was captured with GPS location services enabled.

  2. Install and Open Exifreader

    Download and install Exifreader. Launch the application and use File → Open to load the sample image.

  3. Read EXIF Tags

    The software will display all embedded metadata. Record the following tags: DateTimeOriginal, Make, Model, GPS Latitude, GPS Longitude, Software.

  4. Map GPS Coordinates

    Copy the GPS latitude and longitude values. Open Google Maps and paste the coordinates to verify the geographic location where the photo was taken.

  5. Use ExifTool for Batch Extraction

    Open Command Prompt and run ExifTool to extract metadata from all images in a directory:

    exiftool -csv *.jpg > exif_report.csv

  6. Analyse Embedded Thumbnail

    Use Exifreader to extract the embedded thumbnail image. Compare it to the main image — discrepancies may indicate the image has been edited after capture (anti-forensic indicator).

  7. Document All Metadata

    Record all extracted EXIF fields in the lab report with their values and forensic significance.

📝 Observations & Result

The EXIF data extracted from the sample images revealed the exact date, time, and GPS location of capture, along with the device make and model. The embedded thumbnail matched the main image, indicating no post-capture editing. ExifTool successfully produced a structured CSV report suitable for forensic documentation.

Result: EXIF metadata was successfully extracted from image files using Exifreader, demonstrating the significant forensic value of image metadata in locating suspects and establishing timelines.
