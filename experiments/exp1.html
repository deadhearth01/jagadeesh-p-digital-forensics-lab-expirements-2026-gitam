EXPERIMENT 1

Recovering the deleted files using forensic tools

Aim

To recover deleted files from a storage device using digital forensic tools like Recuva and Autopsy.

Theory

When a file is deleted from a computer, the operating system does not actually erase the data from the storage device. Instead, it simply marks that space as available for new data. The actual file content remains on the disk until it gets overwritten by new files.

This is why forensic tools can recover deleted files. They scan the storage device at the sector level and look for file signatures (also called magic bytes) that identify the start of different file types. For example, JPEG files start with the bytes FF D8 FF.

Recuva is a free file recovery tool that can recover deleted files from hard drives, USB drives, and memory cards. It shows which files are recoverable and which have been partially overwritten.

Autopsy is a digital forensics platform used by law enforcement. It can analyze disk images and recover deleted files along with their metadata like timestamps and original file paths.

Tools Required

  • Recuva (free version)
  • Autopsy Digital Forensics Platform
  • A USB drive with deleted test files
  • Windows PC

Procedure

  1. Copy some test files (images, documents, text files) to a USB drive
  2. Delete the files and empty the Recycle Bin
  3. Open Recuva and select the USB drive
  4. Run a scan and note down the recoverable files
  5. Recover the files to a different location
  6. Now open Autopsy and create a new case
  7. Add the USB drive as a data source
  8. After the scan completes, go to Deleted Files section
  9. Export the recovered files
  10. Compare the results from both tools

Result

The deleted files were successfully recovered using both Recuva and Autopsy. Recuva recovered 5 out of 6 deleted files completely. Autopsy also detected the same files and provided additional metadata like original timestamps and file paths.

This experiment demonstrates that simply deleting files is not enough to permanently remove data from a storage device. Forensic tools can easily recover deleted files unless the storage space has been overwritten.
