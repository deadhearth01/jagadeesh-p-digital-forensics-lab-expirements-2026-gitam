EXPERIMENT 6

Extraction of the browser artefacts

Aim

To extract and analyze browser artifacts including browsing history, cookies, cache, and downloaded files for forensic investigation.

Theory

Web browsers store a significant amount of user activity data locally on the computer. This data, known as browser artifacts, includes browsing history, search queries, cookies, cached files, bookmarks, saved passwords, and download history.

Different browsers store their data in different locations and formats. Google Chrome stores data in SQLite databases located in the user's AppData folder. Mozilla Firefox also uses SQLite databases but in a different location. Microsoft Edge stores data similarly to Chrome since it is also Chromium-based.

For forensic investigators, browser artifacts can reveal what websites a user visited, when they visited them, what they searched for, what files they downloaded, and their login activities. This information can be crucial evidence in many types of investigations.

Tools Required

  • DB Browser for SQLite (free database viewer)
  • NirSoft BrowsingHistoryView
  • Chrome/Firefox/Edge browser data files
  • Windows File Explorer

Procedure

Locating Browser Data:

  • Chrome: C:\Users\[Username]\AppData\Local\Google\Chrome\User Data\Default\
  • Firefox: C:\Users\[Username]\AppData\Roaming\Mozilla\Firefox\Profiles\
  • Edge: C:\Users\[Username]\AppData\Local\Microsoft\Edge\User Data\Default\

Steps:

  1. Navigate to the browser data folder using File Explorer (enable Show Hidden Items)
  2. Copy the relevant database files (History, Cookies, Login Data) to a working folder
  3. Open DB Browser for SQLite
  4. Load the History file and browse the 'urls' table to see visited websites
  5. Check the 'downloads' table to see downloaded files
  6. Load the Cookies file to examine stored cookies
  7. Use BrowsingHistoryView for a more user-friendly view of all history
  8. Export the data to CSV or HTML format for documentation
  9. Analyze the timestamps to create a timeline of user activity

Result

Browser artifacts were successfully extracted from the system. The browsing history, download records, and cookies were retrieved by accessing the browser's SQLite database files. This information provides a detailed record of the user's web activity and can serve as important evidence in forensic investigations.
