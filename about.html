About This Course

Digital Forensics is a branch of forensic science focused on the recovery and investigation of material found in digital devices. This course covers the fundamental concepts and practical techniques used in digital forensic investigations.

Through hands-on laboratory experiments, students learn to use industry-standard tools for data recovery, disk imaging, network analysis, and evidence collection. The skills developed in this course are essential for careers in cybersecurity, law enforcement, and IT security.

Course Objectives

  • Understand the fundamentals of digital forensics and its legal aspects
  • Learn to use forensic tools for data acquisition and analysis
  • Develop skills in recovering deleted files and hidden data
  • Analyze network traffic for security investigations
  • Document and present forensic findings professionally

Tools Covered

  • Autopsy - Open-source digital forensics platform
  • FTK Imager - Forensic disk imaging tool
  • Recuva - File recovery software
  • Wireshark - Network protocol analyzer
  • Sysinternals - Windows system utilities
  • Exifreader - EXIF metadata extraction
